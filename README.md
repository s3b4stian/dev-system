# Dev System

Scripts for install dev environment.

```
git clone https://github.com/s3b4stian/dev-system.git
```

Environment config
```
chmod +x ./dev-system/start.sh
./dev-system/start.sh
```

Samba installing and config
```
chmod +x ./dev-system/samba.sh
./dev-system/samba.sh
```

Netplan config
```
chmod +x ./dev-system/netplan.sh
./dev-system/netplan.sh
```

Install PHP basic dev tools
```
chmod +x ./dev-system/php_tools.sh
./dev-system/php_tools.sh
```
